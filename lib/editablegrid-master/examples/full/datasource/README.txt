This directory contains the XML data source for our grid:
- demo.csv: CSV file containing sample data (500 lines)
- demo.php: PHP script that generates the JSON or XML data on-the-fly from this CSV file
- demo.xml: contains the XML data that has been generated by demo.php, so that non PHP users can also run the examples
- demo.json: contains the JSON data that has been generated by demo.php, so that non PHP users can also run the examples
